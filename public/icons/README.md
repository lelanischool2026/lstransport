# Icon Files Placeholder

This directory should contain PWA icon files in the following sizes:

- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

You can generate these icons from your logo using tools like:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

The icons should:
1. Be square (same width and height)
2. Have a transparent or solid background
3. Be recognizable at small sizes
4. Follow the school's branding colors
